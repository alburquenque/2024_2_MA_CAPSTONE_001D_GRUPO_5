Al escanear un producto procura tener en consideración los siguientes pasos:

1.- La aplicación posea permisos para utilizar la cámara.

3.- Una vez escaneado el producto elegido puedes agregalo a tu carrito.

4.- Dentro del carrito podrás ver el producto que agregaste.

5.- Presiona "Continuar compra" para realizar el proceso de pago.
